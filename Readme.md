A clean, responsive, and interactive classic Rock-Paper-Scissors game built using native web technologies. This project was developed as part of foundational web development practice.

## ✨ Features
* **Interactive UI:** Smooth transitions and visual feedback for player selections.
* **Instant Score Tracking:** Automatically calculates and displays player vs. computer scores.
* **Responsive Design:** Fully optimized to play seamlessly on desktops, tablets, and smartphones.
* **Smart AI:** Randomized computer choices ensuring fair gameplay.

## 🛠️ Built With
* **HTML5:** Semantic structure for the gaming canvas and controls.
* **CSS3:** Modern styling, layout alignments, and hover animations.
* **JavaScript (ES6):** Core game logic, state management, and dynamic DOM manipulation.

## 🎮 How to Play
1. Click the **Live Demo** link or open `index.html` locally in any browser.
2. Select your weapon of choice by clicking either **Rock**, **Paper**, or **Scissors**.
3. The computer will instantly make a random selection.
4. The screen will display the round winner and update the scoreboard.
5. Rules follow standard logic:
   * 🪨 **Rock** crushes ✂️ **Scissors**
   * 📄 **Paper** covers 🪨 **Rock**
   * ✂️ **Scissors** cuts 📄 **Paper**

## 🔧 Installation & Local Setup
To run this project locally on your machine:
1. Clone this repository: `git clone https://github.com`
2. Navigate to the project directory.
3. Open `index.html` in any web browser.
